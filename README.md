# Tom & Gina · runlevel-labs.github.io

Private GitHub-Pages-Website von Tom & Gina.

Die Seite sammelt kleine Web- und App-Projekte, technische Experimente, KI-gestützte Inhalte und visuelle Konzepte rund um Runlevel Labs, runlevel.eth und Gina.

## Inhalt

- statische Website mit HTML, CSS und wenig JavaScript
- Projektübersicht
- Gina Universum
- Datenschutz- und Impressumsseiten
- Audio-/Video- und Bildmaterial
- GitHub-Pages-Deployment über `main` und `/ (root)`

## Technik

Dieses Repository ist bewusst einfach gehalten:

- `index.html` – Startseite
- `style.css` – gemeinsames Styling
- `gina.html` – Gina Universum
- `datenschutz.html` – Datenschutzhinweise
- `impressum.html` – Impressum
- `notizen-datenschutz.html` – Datenschutzseite für die Android-App „Notizen“
- `robots.txt` und `sitemap.xml` – Suchmaschinen-Grundstruktur

Keine Build-Tools, kein Framework, kein Tracking.
